Welcome to route1io-python-connectors's documentation!
======================================================

At `route1.io <https://www.route1.io>`_ we've written high level, easy-to-use abstractions in Python for
connecting to common APIs used in marketing analytics. This repository contains
our official code, documentation, and tutorials to help you on your way accessing
what matters most: *your data*.
